Check Box

No comando input utilize a opção checkbox no atributo type, os atributos name e value
são para dar nome aquelas checkbox e retornar um valor, vale lembrar que um grupo de
checkbox de ter o mesmo nome.
Exemplo: input name="teste" value="t" type="checkbox">teste
Checkbox pode ter mais de uma opção selecionada!

teste teste2 teste3

Radio Buttons

Só pode ter um selecionado, seu tipo é radio, segue a mesma estrutura do checkbox
Exemplo :input name="sexo" value="m" type="checkbox">Masculino

Exemplo de Label

Usando a label é possível que ao clicar no nome da descrição do item, ele também é selecionado
Exemplo: label for="ts">input id="ts" name="teste" value="t" type="checkbox">teste /label>


Drow List

Use o comando select e crie as opções a serem selecionadas com o comando option.

File inputs

Apenas inserir no type a opção file, quando File Inputs estiverem dentro de um forms,
deve-se usar o atributo enctype com a opção multipart/form-data, para indicar que o forms
vai receber tanto arquivo texto como arquivo data
